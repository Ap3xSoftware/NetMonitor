📌 Acerca del ProyectoNetMonitor Pro es una solución empresarial de monitoreo de infraestructuras IT de alto rendimiento. Diseñado desde cero para esquivar los cuellos de botella tradicionales de las bases de datos relacionales durante el streaming de video, NetMonitor permite observar docenas de computadoras en tiempo real de forma simultánea utilizando una fracción de los recursos de hardware habituales.Ideal para laboratorios de informática, oficinas corporativas, call centers y entornos de seguridad que requieren auditoría visual en vivo.✨ Características Principales🚀 Streaming Optimizad: Motor de transmisión de video P2P-like que omite escrituras en base de datos, logrando 100+ conexiones activas con menos del 5% de impacto en CPU del servidor.📊 Métricas en Vivo: Monitoreo preciso del uso de CPU, Memoria RAM y almacenamiento en Disco de cada terminal conectada.🗂️ Gestión por Grupos: Organización dinámica de computadoras (ej. Ventas, Laboratorio A, Gerencia) con actualización AJAX instantánea.🕹️ Control Remoto: Ejecución de comandos del sistema a distancia (ej. Bloqueo de sesión inmediato).👻 Agente Invisible: Agente cliente escrito en Python compilado (.exe), que se ejecuta silenciosamente en segundo plano sin interfaces que distraigan al usuario.📱 Panel Responsive: Dashboard administrativo estilizado con Tailwind CSS, adaptable a cualquier pantalla o muro de monitoreo (Video Wall).🏗️ Arquitectura del SistemaEl sistema opera bajo una arquitectura cliente-servidor de alta eficiencia:Agente Cliente (Python): Utiliza bibliotecas de bajo nivel (mss, psutil) para capturar la pantalla y telemetría, empaquetando todo en memoria binaria y transmitiendo por protocolo HTTP asíncrono.Backend (PHP/Apache): Recibe las transmisiones directas y actualiza los punteros de memoria, mientras que la telemetría se registra en MariaDB.Frontend (JavaScript/Tailwind): Un panel SPA (Single Page Application) falso que se actualiza sin recargar mediante polling inteligente.🖼️ Interfaz (Previews)Nota: Agrega capturas de pantalla de tu panel aquí.Dashboard PrincipalVista en Vivo````💻 Requisitos del SistemaPara el Servidor (Recomendado: Ubuntu Server 22.04+ / aaPanel)Servidor Web (Apache o Nginx)PHP 8.0 o superiorMySQL 8.0 o MariaDB 10.x2GB RAM mínimo (Se recomienda 4GB para entornos > 100 PCs)Para los Clientes (Windows 10/11)Permisos de ejecución del archivo agente.exeConexión a la red local (LAN/WLAN)🛒 Adquisición y Licencia ComercialEste es un software de grado comercial cerrado.
